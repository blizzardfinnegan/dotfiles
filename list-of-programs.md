# List of Necessary Programs

## Strictly necessary

- OpenSSH
- GnuPG
- NeoVim
    - NeoVim requires LuaJIT, which has a RISC-V port available. See [here](https://github.com/plctlab/LuaJIT/tree/v2.1-riscv64-pr)
- fish
- tmux
- curl || wget
- git
- rsync

## Dev tools

- rustup 
- C++ dev platform (Clang AND LLVM, GCC, CMake, Ninja)
- gradle
- >=jdk17

## Monitoring tools

- btop

## Misc.

- Links2/elinks/qutebrowser
    - firefox for higher end hardware
